# diversity-plugins


Hexo Diversity 主题插件集合，旨在为 Hexo 博客提供更多功能和定制选项。

## 插件列表

- [hexo-hot-config](./packages/hexo-hot-config/README.md)  - Hexo 热配置插件，支持实时更新配置

## 安装

每个插件都可以单独安装，请参考各插件目录下的 README 文件获取详细安装和使用说明。

## 贡献

欢迎提交 Pull Request 或创建 Issue 来帮助改进这些插件。
